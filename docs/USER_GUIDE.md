# CASSANDRA AI — 사용 설명서

> Toss × DART × LLM 리스크 모니터링
> 
> **코스닥 시장의 소음 속에서 이상 징후 신호를 찾아내는 공익 목적의 분석 도구**

---

## 누가 사용하나요?

### 📈 주식 투자자
한계기업의 공시 패턴을 분석하여 투자 위험을 사전에 감지합니다.

### 📰 증권 기자
기업·인물 간 숨은 연결고리를 찾아 심층 보도의 근거 자료로 활용합니다.

### 👮 검찰 / 경찰
자본시장 불공정거래 조사에 필요한 공시 기반 관계망 정보를 제공합니다.

---

## 기능별 사용법

### 1. 관계망 분석 — "누가 누구와 연결되어 있나?"

**사용 시나리오**
- "CBI의 최대주주는 누구이며, 이 사람이 또 어떤 회사에 관여했나?"
- "신승수라는 인물이 등장하는 공시는 어디인가?"

**사용 방법**
1. 검색창에 회사명 또는 인물명 입력
2. 관계망 그래프에서 노드 클릭 → 상세 정보 확인
3. 핀보드에 관심 대상 고정 → 리포트 생성

**투자자 관점**: CB 발행·대주주 변경이 반복되는 기업을 사전 식별
**기자 관점**: 동일 인물이 다수 기업에 등장하는 패턴 포착
**검찰/경찰 관점**: 페이퍼컴퍼니-실소유자 연결고리 추적

### 2. DART 분석 챗봇 — "공시 내용을 요약해줘"

**사용 방법**
1. 우측 챗봇에 회사명 입력 (예: "딥커머스")
2. 기간 선택 (1개월~36개월)
3. 분석 결과 확인: 카테고리별 공시 건수 + 위험 신호

**출력 예시**
```
⚠️ 증자/감자 7회 — 빈번한 자본 변동
⚠️ 매매정지 4회 — 주식병합·분할 반복
```

### 3. 경제 지표 대시보드 — "지금 시장에서 무슨 일이?"

**사용 방법**
1. 상단 메뉴 → 경제 지표
2. 일일 고위험 시그널 확인 (8종 룰셋 자동 분석)
3. 사명변경·대주주변경·소송·CB 발행 필터

**투자자 관점**: 거래량 급증 + 공시 이벤트 동시 발생 종목 포착
**기자 관점**: "오늘의 고위험 기업" 리스트로 취재 대상 선정

### 4. 인물 검색 — "이 사람, 다른 회사에도 있나?"

**사용 방법**
1. 상단 메뉴 → 인명검색
2. 이름 입력 + 기간 선택 (1년/3년/5년)
3. DB 결과 없으면 "DART 공시통합검색에서 찾기" 클릭

**검찰/경찰 관점**: 특정 인물의 전체 커리어 이력 조회
**기자 관점**: 취재 대상 인물의 숨은 연관 기업 발견

### 5. 주식셀럽 WIKI — "주요 투자자 정보"

**사용 방법**
1. 상단 메뉴 → WIKI
2. 10명 주요 투자자 프로필 확인
3. 코멘트 기능으로 집단 지성 참여

### 6. 제보·분석 게시판 — "분석 요청하기"

**사용 방법**
1. 상단 메뉴 → 제보·분석
2. "분석을 원하는 기업, 개인을 입력하세요" → 등록
3. 처리 완료 → 게시판에서 결과 확인

### 7. 증거 자료 수집 (검찰/경찰)

1. **관계망 그래프**: 인물-회사-조합 연결고리 시각화
2. **인물 이력**: DART 지분공시 기반 선임/사임/보유 이력
3. **리포트 다운로드**: MD 형식으로 증거 자료 보존
4. **CASSANDRA AI는 평가하지 않습니다**: 모든 데이터는 DART 공시 원문 기반

---

## 법적 주의사항

1. 본 서비스는 DART 공시 사실의 색인·분석 도구입니다
2. 특정 개인·법인에 대한 평가가 아닙니다
3. 투자 조언이 아닙니다. 모든 투자 판단은 이용자 본인의 책임입니다
4. 생성형 AI의 Hallucination 가능성이 있습니다
5. 모든 데이터는 DART 접수번호(rcept_no)로 역추적 가능합니다
